Accessibility Statement for https://www.tabithariley.digital/

This accessibility statement applies to the www.tabithariley.digitial website.

This website is run by Tabitha Riley. We want this site to be accessible to as many people as possible. For example, that means you should be able to:

  1. zoom in up to 500% and the text does not spill off the screen.
  2. navigate the website using just a keyboard if required.
  3. toggle the navigation menu with mouse hover or button click, but all information from the navigation is available without it.

How Accessible this Website Is

Though we have strived to develop this site with extensive accessibility, we are aware that some elements of this site are not fully accessible:

  1. The navigation menu spills off the screen when zoomed in past 150% (depending on screen size), but there is a scroll bar for this scenario.
  2. You cannot modify line height or letter spacing of text.
